# STEP 1 – Full-stack base (senza AI)

**Livello:** Facile
**Obiettivo:** Capire come parlano i pezzi

## Mini-progetto: Web App “Hello Data”

### Architettura

Frontend (React) -> Node.js (API) -> Python (Processing)

### Cosa fare:

1. **Frontend**: Crea un form dove l'utente inserisce dei numeri. Visualizza il risultato.
2. **Node.js**: Crea un endpoint REST che riceve i dati dal frontend e li inoltra a Python.
3. **Python**: Crea una semplice API che riceve i numeri e restituisce:
   - Somma
   - Media
   - Max / Min

### Teoria:

- HTTP, REST
- JSON
- Client ↔ Server
- Separazione Frontend / Backend
- Node come “ponte”
